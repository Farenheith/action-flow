A primeira pessoa a ser incluída é sempre o Beneficiário. No entanto, a figura do beneficiário muda de acordo com o tipo de caso que você deseja incluir. Abaixo segue a relação:



  • Solicitação de busca / Tracing Request: O beneficiário é a pessoa buscada;

  • Mensagem Cruz Vermelha / Red Cross Message (RCM): O beneficiário é o destinatário;

  • Mensagem São e Salvo / Safe and Well Message: O beneficiário é o destinatário;

  • Salamat / Oral Greetings: O beneficiário é o destinatário;

  • Chamada telefônica / Phone Call completed: O beneficiário é a pessoa que solicitou a chamada;

  • Documentos / Documents: A pessoa objeto da documentação. Isso é importante diferir, pois se o atendido solicita documentos que dizem respeito a ele, ele é o beneficiário. Se é solicitado documentos que dizem respeito a outra pessoa, essa pessoa é o beneficiário;
