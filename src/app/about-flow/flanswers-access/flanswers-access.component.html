Há dois sites diferentes para o FL ANSWERS: o site de treinamento e o site de produção.

O site de treinamento é destinado aos voluntários que estão aprendendo a mexer na ferramenta. Lá vocês poderão estudar o Sistema e executar os exercícios para se habilitarem.

O ambiente de produção é o oficial do FL ANSWERS onde lançaremos os casos reais. Você só deve entrar neste ambiente se já recebeu a permissão do seu coordenador para fazê-lo! Se você quiser tirar dúvidas sobre o uso da ferramenta, por exemplo, mesmo após já ter permissão, acesse o ambiente de treinamento.

É importante se certificar que, ao inserir o link no seu navegador, ele comece com https. Se começar com http ou sem nenhum dos dois ele não irá funcionar.

O link de treinamento é: https://bra.FL ANSWERSprep.org/.

O link de produção é: https://bra.FL ANSWERS.org/.

ATENÇÃO: Caso você esteja treinando e o cabeçalho do sistema estiver vermelho, pare tudo o que você está fazendo! Você está no ambiente de produção do FL ANSWERS!
