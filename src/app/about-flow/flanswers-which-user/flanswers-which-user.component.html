Os usuários do FL ANSWERS são criados sempre utilizando o seu e-mail institucional. Para o primeiro acesso você recebe o usuário com uma senha temporária neste e-mail e deverá alterá-la na primeira utilização. A própria ferramenta irá pedir isso automaticamente para você.

Caso você não tenha recebido o e-mail, cheque se seu usuário já foi solicitado. Se sim, você pode acessar o link https://myaccount.FL ANSWERS.org/accounts/Reset e solicitar o reset de sua senha. É preciso saber o nome do usuário, no entanto. Você pode conversar com o líder de organização e gerenciamento ou com o coordenador para saber.

Com isso, você receberá um link em seu e-mail institucional para fazer esse procedimento. Se o e-mail não chegar, entre em contato com o seu coordenador para entender o motivo.
